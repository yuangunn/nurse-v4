근무표 작성 순서

① 설정

간호사 등록 → 요일별 필요인원 설정 → 규칙 관리

② 사전입력

주휴(주) → 연차(V) → 생리휴가(생) → 희망 근무 순으로 입력. 빈 셀은 솔버가 자동 배정합니다.

③ 분석

인원 과부족 확인 → 주휴 추천 배분 → "사전입력에 적용" 클릭

④ 스케줄 생성

"생성" 버튼 클릭 → 솔버가 최적 근무표 자동 생성 (5~20분 소요)

⑤ 확인 & 수정

생성 결과 확인 → 셀 클릭으로 수동 수정 → CSV/인쇄로 내보내기

⚙️ 설정 탭

  • 간호사 추가/수정/삭제 (드래그로 순서 변경)
  • CSV 템플릿 다운로드 → 엑셀에서 편집 → 일괄 가져오기
  • 요일별 D/E/N 필요 인원 설정
  • 하드 제약 (연속 근무, 역순 전환 금지 등)
  • 소프트 배점 (순방향 전환 보상, 야간 공평성 등)

✏️ 사전입력 탭

  • 셀 클릭 → 근무 선택 팝업
  • 키보드 직접 입력: D/E/N/V + 방향키 이동
  • 드래그 선택: 여러 셀을 드래그 후 한 번에 채우기
  • 주휴 클릭 시 자동 배분 옵션 (로테이션/고정/이번만)
  • 날짜 우클릭: 법정공휴일 지정

📊 분석 탭

  • 일자별 D/E/N 필요인원 + 여유 히트맵
  • 주휴 추천 배분 → "사전입력에 적용" 한 번에 반영
  • 경고: 인원 부족/빡빡한 날짜 표시

📅 스케줄 탭

  • 오차(MIP Gap) 조절 — 정밀(0%)~10%
  • 사전입력 완화 / 주휴 무시 / V 무제한 옵션
  • 완화 설정: 휴가/근무/쉬는날 유지 보너스 조절
  • 생성 후 셀 클릭으로 수동 수정 (빨간 테두리 표시)
  • Charge 숨기기 / 근무별 색상 토글

💾 저장 탭

  • 생성된 스케줄 저장/불러오기/삭제
  • 생성 완료 시 자동 저장

❓ 근무표 생성이 실패합니다 (Infeasible)

1. 사전입력 완화 체크 → 재시도
2. 실패 시 주휴 무시 추가
3. 실패 시 V 무제한 추가
4. 그래도 실패 시 에러 메시지의 원인 확인 → 해당 규칙 완화 (설정 탭)

❓ 사전입력 완화란?

사전입력한 근무를 솔버가 변경할 수 있게 허용합니다. 휴가(V/생)는 높은 보너스로 거의 유지되고, 일반 근무(D/E/N)는 유연하게 교체됩니다. "완화 설정"에서 보너스를 조절할 수 있습니다.

❓ 생성 시간이 너무 오래 걸립니다

오차를 2%~5%로 올리면 훨씬 빠르게 생성됩니다. 정밀(0%)은 최적해를 찾으려 시간이 오래 걸립니다.

❓ 간호사를 한번에 등록하려면?

설정 탭 → "📥 템플릿" 다운로드 → 엑셀에서 편집 → "📤 가져오기"로 업로드

❓ 다크 모드는 어디서?

사이드바 하단 "다크 모드" 버튼을 클릭하세요.

❓ 데이터는 어디에 저장되나요?

모든 프로필과 DB는 %APPDATA%\NurseScheduler\에 저장됩니다. 앱을 삭제해도 데이터는 보존됩니다.
